# ZTO Express（中通快递）

中通快递助手：查物流、估运费、找网点、解决寄件问题

## What it does
- 中通快递单号物流轨迹查询与状态解读
- 中通寄件费用估算（按重量/距离/服务类型）
- 附近中通网点查找、寄件指引与异常件排查

## Example scenarios

**1. 查快递物流**
> 👤 帮我查一下中通快递单号 773012345678901
> 🤖 显示运单最新节点（如"包裹已到达上海分拣中心"）、始发地和目的地、预计送达时间、异常备注。如单号格式不对，提示用户核实

**2. 估算寄件费用**
> 👤 从广州寄到北京，一个3公斤的普通包裹，中通多少钱？
> 🤖 根据重量和线路给出费用估算，解释中通标准定价因素（首重价格、续重费用、偏远地区附加费）。如无精确数据，给出谨慎的价格区间并说明假设条件

**3. 寄件物流异常排查**
> 👤 我在淘宝买的东西用中通发的，好几天没更新物流了，帮我看看是不是丢了？
> 🤖 读取运单号完整物流轨迹，判断是否超过48小时未更新等异常状态。分析可能原因（转站未扫描、节假日延迟等），给出针对性建议：联系卖家催件、打中通客服95311投诉、在菜鸟App发起物流查询
